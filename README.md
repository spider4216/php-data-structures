# PHP Data Structures

This package provide a general data structures for PHP programming language like:

- Array
- Vector
- Linked List
- Queue
- Stack
- Binary Tree

All of that contain base algorithm for operations with each data structure like:

- Insert operation
- Delete operation
- Find operation
- Sort operation

Many of operations covered by unit tests
